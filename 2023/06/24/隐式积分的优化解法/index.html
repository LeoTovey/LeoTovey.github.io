隐式积分的优化解法

Posted on 2023-06-24

introduction

图形学中，我们很难用更大的时间步长来模拟隐式积分。最主要的原因就是牛顿法的不稳定性，它不能覆盖足够大的时间步长。

为了解决这个问题，论文把求解非线性系统方程转化为一个优化问题。

minimization leads to a method that is both simpler and faster than the equivalent nonlinear root-finding problem

2. 时间积分

根据后向欧拉法： \[ \frac{x^{n+1}-x^n}{\Delta t}=v^{n+1} \quad M \frac{v^{n+1}-v^n}{\Delta t}=f^{n+1}=f\left(x^{n+1}, v^{n+1}\right) \] Eliminating \(v^{n+1}\) yields \[ M \frac{x^{n+1}-x^n-\Delta t v^n}{\Delta t^2}=f\left(x^{n+1}, \frac{x^{n+1}-x^n}{\Delta t}\right) \] 然后定义

\[ h\left(x^{n+1}\right)=M \frac{x^{n+1}-x^n-\Delta t v^n}{\Delta t^2}-f\left(x^{n+1}, \frac{x^{n+1}-x^n}{\Delta t}\right) \] 我们的非线性问题就转化为求解\(h(x^{n+1})=0\)的根

To do this, one would start with an initial guess \(x^{(0)}\), such as the value predicted by forward Euler. This estimate is then iteratively improved using the update rule \[ x^{(i+1)}=x^{(i)}-\left(\frac{\partial h}{\partial x}\left(x^{(i)}\right)\right)^{-1} h\left(x^{(i)}\right) . \]

  • 如果方程h足够好，同时初始猜测足够接近最终的解，那么牛顿法可以很快地收敛。但是如果初始值设置不恰当，那么牛顿法就会收敛缓慢甚至不收敛

  • 对于足够小的步长来说，前向积分和后向积分是差不多的。因此可以使用前向积分来提供一个比较好的初始值。

  • 对于相对大的步长来说，前向积分就不稳定，因此不能使用前向积分的结果来作为初始值。

  • 牛顿法在大的时间步长下就不会收敛

  • 在有些情况下，我们会采用固定迭代次数的牛顿法来进行模拟，而不是一直迭代到收敛到一定范围。这样可以从隐式方法中得到足够多的好处，因为少量的牛顿法迭代的时间步长就已经比显式方法高几个量级了。

2.1 假设

为了使得解决方法更加鲁棒，我们需要提前做出一些假设

A1: Masses are positive

质量矩阵是对称和正定的

A2: \(\quad f=-\frac{\partial \Phi}{\partial x}\) for some function \(\Phi\)

保守力一般是满足这个条件的，绝大部分弹力模型也满足这个条件。

（一个粒子从初始位置A移动经过任意闭合路径后，又回到位置A ，而此作用力所做于粒子的机械功都等于零，则此作用力满足保守力的条件，可以被分类为保守力。例如重力、弹簧力、电场力。非保守力包括摩擦力，空气阻力）

A3: \(\quad \Phi\) is bounded from below

有下界限

A4: \(\quad \Phi\) is \(C^1\)

2.2 最小化问题

The solution to making Newton’s method converge reliably is to recast the equation solving problem as an optimization problem, for which robust and efficient methods exist.

首先我们把\(h\left(x^{n+1}\right)=M \frac{x^{n+1}-x^n-\Delta t v^n}{\Delta t^2}-f\left(x^{n+1}, \frac{x^{n+1}-x^n}{\Delta t}\right)\) \[ h(x)=M \frac{x-x^n-\Delta t v^n}{\Delta t^2}+\frac{\partial \Phi}{\partial x} . \] We note that if we set \[ \hat{x}=x^n+\Delta t v^n \quad E(x)=\frac{1}{2 \Delta t^2}(x-\hat{x})^T M(x-\hat{x})+\Phi \] 如果上一节中提到的假设都是成立的，那么E的全局最优是存在的。

我们就把问题转化为了一个优化问题，对于 E 的任意局部最优点都是原非线形系统的解

3. 优化

整个流程

Our optimization routine begins with an initial guess, \(x^{(0)}\) Each iteration consists of the following steps: 1) \(\star\) Register active set 2) Compute gradient \(\nabla E\) and Hessian \(H\) of \(E\) at \(x^{(i)}\) 3) Terminate successfully if \(\|\nabla E\|<\tau\) 4) Compute Newton step \(\Delta x=-H^{-1} \nabla E\) 5) Make sure \(\Delta x\) is a downhill direction 6) Clamp the magnitude of \(\Delta x\) to \(\ell\) if \(\|\Delta x\|>\ell\) 7) Choose step size \(\alpha\) in direction \(\Delta x\) using a line search 8) Take the step: \(\boldsymbol{x}^{(i+1)}=\boldsymbol{x}^{(i)}+\alpha \Delta \boldsymbol{x}\) 9) \(\star\) Project \(x^{(i+1)}\)

6. mpm

利用上文中提到的优化思路，我们可以定义 \[ E\left(v_i\right)=\sum_i \frac{1}{2} m_i\left\|v_i-v_i^n\right\|^2+\Phi\left(x_i^n+\Delta t v_i\right), \] where \(m_i\) is the mass assigned to grid index \(i\).

Our final \(v_i^{n+1}\) is computed so that \(E\left(v_i^{n+1}\right)\) is minimized.

We solve this minimization problem as in Section 3. Note that we apply plasticity explicitly as in the original formulation.

Using larger time steps causes our linear systems to become slower to solve.

In the case of MPM, we found it beneficial to use the diagonal preconditioner \[ L_{i i}=\sum_p \operatorname{diag}\left(m_p w_{i p} I+\Delta t^2 V_p^0 H\right), \] where \[ \boldsymbol{H}=\left(\lambda_p+\mu_p\right) \nabla w_{i p} \nabla w_{i p}^T+\mu_p \nabla w_{i p}^T \nabla w_{i p} \boldsymbol{I} . \]
